# Public toot viewer

マストドンの公開タイムラインをみるやつ。inspired by https://junk.azyobuzi.net/mastodonptl/

https://m6n-view.hnle.tk/

## Build Setup

``` bash
# install dependencies
yarn install

# serve with hot reload at localhost:3000
yarn dev

# build for production with minification
yarn build

# run tests
yarn test
```
